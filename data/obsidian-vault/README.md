# TroughTrader Knowledge Base

> "For traders who are always in the trough."

This Obsidian vault tracks all development changes, feature updates, trading notes, and architecture decisions for the TroughTrader project.

## Structure

- **Development/** — Engineering notes, feature specs, changelogs
  - **Features/** — Individual feature documentation
  - **Daily Notes/** — Auto-generated git commit logs
- **Trading/** — Market analysis and trading strategies
  - **Strategies/** — Trading strategy notes
  - **Notes/** — General trading observations
  - **Market Analysis/** — Sector/stock research
- **Architecture/** — System design and data architecture decisions

## Auto-Update

Every git commit triggers a hook that appends to `Development/Daily Notes/YYYY-MM-DD.md`.

Open this vault in Obsidian: **File → Open vault → `data/obsidian-vault`**
